소전 일러 추출 프로세스 // 24.05.12 개정
1. 소전 파일 추출기 실행해서 asset 파일 다운로드

2. 추출 폴더 검색 키워드
"character" --spine --npc
검색된 파일들 1char 폴더로 이동

@@푸른 바다 위 검은 돛 이후 이미지는 hd 파일 없이 일반이 2k

3. gfuf 실행--1char 폴더 선택--해당 폴더 내 ab 파일들이 1update 폴더에 있는지 확인
있으면 스킵, 없으면 1update 폴더로 이동 후 새로운 파일은 1brandnew 폴더에 복사

4. 1brandnew 폴더 확인해서 gffc 업데이트

5. 1brandnew 폴더 exe 파일로 드래그 하여 파일 추출

6. gffc 실행-선택한 폴더내 이미지를 돌면서(for) 파일명 (이름) == 사전 인형명 비교
이미지 파일명을 이름-(스킨번호)-(A:검열)-(D:중상) 으로 변경
변경한 파일들을 모아 result 폴더를 생성해 저장

7. 검수 후 보관소에 적용
